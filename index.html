📚 사고력 OS 완전 가이드

이 앱은 ChatGPT API를 활용하여 독서 사고력을 12단계로 심층 분석합니다.
책을 읽는 것을 넘어 생각하는 힘을 기릅니다.

1
API 키 설정
상단 설정바에 OpenAI API 키(sk-...)를 입력하고 저장하세요.
2
모델 선택
GPT-4o (추천), 또는 예산에 맞는 모델을 선택하세요.
3
탭 선택
상단 탭에서 원하는 사고 훈련 방식을 선택하세요.
4
내용 입력
책 제목, 문장, 문단, 또는 나의 생각을 입력합니다.
5
결과 저장
분석 결과는 📒 노트저장 버튼으로 독서 노트에 보관하세요.
⚠️ API 키는 브라우저 내부에만 저장되며 외부로 전송되지 않습니다.
